# SOEDI MAMPIR 💈

Website reservasi online untuk Barbershop SOEDI MAMPIR. Website ini memudahkan pelanggan untuk melakukan booking jadwal cukur, memilih kapster favorit, dan melihat daftar layanan yang tersedia.

## 🚀 Fitur Utama
- **Sistem Booking:** Memilih layanan, kapster, dan waktu kedatangan.
- **Integrasi WhatsApp:** Konfirmasi otomatis ke WhatsApp Admin untuk mempermudah proses DP/pembayaran.
- **Dashboard Admin:** Panel khusus untuk pemilik (Bang Fadil) melihat rekap jadwal harian.
- **Responsive UI:** Tampilan yang nyaman diakses lewat HP maupun Desktop.

## 🛠 Teknologi yang Digunakan
- **Frontend:** Next.js, Tailwind CSS.
- **Backend:** Django (Python).
- **Database:** SQLite.

## ⚙️ Cara Menjalankan

### 1. Setup Backend
1. Masuk ke folder backend: `cd backend`
2. Jalankan server: `python manage.py runserver`

### 2. Setup Frontend
1. Masuk ke folder frontend: `cd frontend`
2. Install dependencies: `npm install`
3. Jalankan aplikasi: `npm run dev`
